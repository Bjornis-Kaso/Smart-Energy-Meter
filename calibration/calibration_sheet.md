## Legend

| Symbol | Meaning |
|--------|---------|
| Mx(V/A/P) | Reading from the instrument under test |
| Mo(V/A/P) | Reading from the reference (standard) instrument |
| Ga | Absolute error = Mx − Mo |
| θ | Correction = −Ga |
| Grl (%) | Relative error % = (Ga / Mo) × 100 |
| Grf (%) | Referenced error % = (Ga / Full-scale) × 100 |
